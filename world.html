03 — Kaijin System

怪異について KAIJIN

怪異とは、集合的な噂・言説・信念が臨界点を超えた時に現実へと侵食する存在だ。専門家はこれをバグと呼ぶ。全ての怪異は核とルールを持つ。

Core — 核
核とは何か
怪異の存在の中心であり消滅条件の鍵だ。基本的には物に宿る。特定の場所・道具・建造物・記録媒体などが核になる。稀に死体や生きた人間に宿る場合があり、対処の難度と倫理的複雑さが跳ね上がる。

核を無効化・破壊することが怪異消滅の最短経路だが、どこに何が核なのかは調査と解明を経て初めて分かる。
Rule — ルール
ルールとは何か
怪異が人間を害するための条件だ。ルールを破れば破るほど怪異から逃げられなくなる。

ルールは怪異が人を殺す手順であると同時に、怪異が人を捕捉・追跡するトリガーでもある。無意識にルールを一度も破らなかった一般人が理由も分からず生還するケースがある。

強い怪異ほどルールが複雑で厳密になる傾向がある。
怪異と神について： 討伐者の間ではこういう話がある。「恐れられているか信仰されているかの違いだけだ。」真偽は不明だが、神社の御神体が怪異化したという記録が残っていることは事実だ。神だから討伐できないという保護はこの世界に存在しない。
改造個体と別個体
Modified — 改造個体
ルール変質 50%未満
元の核とルールが主体として残っている。既存の知識がある程度通用するが、ある一点でルールが決定的に食い違う瞬間が来る。調査段階では「文献との齟齬・被害パターンの不一致・核の所在の混乱・目撃証言の分裂」という四つの違和感として現れる。
New Entity — 別個体
ルール変質 50%以上
元の知識はほぼ役に立たない。 完全に別の怪異として扱う。最も恐ろしいパターンは「改造個体と解明して討伐に挑んだ瞬間に別個体だと判明するケース」だ。怪異は健在で討伐者はルールを破り続けた状態で無防備になっている。
04 — Classification

等級体系 GRADE SYSTEM

怪異は存在強度等級と脅威度種別の二軸で表記される。二つの軸は独立しており、必ずしも一致しない。

存在強度等級（特級〜五級）
等級基準特徴
特級現状の手段では消滅が極めて困難封印・管理が限界。鵺ヶ原事変の怪異が該当
一級信念密度が非常に高い強力な怪異長い歴史を持つ。対処法が民俗学記録に存在する場合が多い
二級記録のある古い怪異の大半訓練された討伐者であれば対処可能
三級拡散が進んだ新しい怪異などSNS時代に急成長するケースがある
四級発生したばかりの新しい怪異初見殺しが最大のリスク
五級魔法事故由来の微小怪異自然消滅することが多い
脅威度種別（甲種〜丁種）
種別基準対処の考え方
甲種無差別に人を害する。接触した全員が標的即時対応が必要
乙種ルールを破った者など条件付きで害するルール遵守で生存できる
丙種刺激しなければ安全放置が最善の場合がある
丁種人を害さない。共存可能または信仰対象に近い討伐より封印・共存の選択肢もある
重要な注意： 強いが安全な怪異（一級丁種）、弱いが危険な怪異（五級甲種）が存在する。等級だけで危険度を判断するな。さらに影響範囲（広域型・局所型）と被害性質（物理型・精神型・複合型）が付記される。
永続性による分類
Type A
一時型
核が破壊されると消滅する標準的な怪異。ただし噂の中身が変質して似た性質の派生体が後日発生するケースがある。派生体は元の怪異と別個体として扱え。
Type B
循環型（都市伝説型）
社会に深く根を張った怪異。核を破壊しても噂が生き続ける限り新たな核を形成して再出現する。討伐は根絶ではなく制圧だ。
Type C — 破壊困難
定着型（土地型）①
核が土地と一体化しすぎており通常手段では無効化できない。大規模な魔法での破壊はさらに大きな怪異を生むジレンマがある。公的機関が封鎖して放置する理由がここにある。
Type C — 一度限り
定着型（土地型）②
核の特定と破壊に成功すると噂の構造ごと崩壊して完全消滅する。達成した討伐者は伝説扱いされる。白銀の廃線事件はこのタイプへの対処事例として語り継がれている。
05 — Exorcism Process

討伐プロセス PROCESS

討伐は三段階のプロセスで構成される。戦闘はあくまで最終段階であり、その前の二段階が生死を分ける。

01
調査プロセス
怪異の存在を確認し正体と性質を特定する段階。

明らかにすること：この怪異は何者か、古い怪異か新しい怪異か、核はどこに宿っているか、被害のパターンと範囲はどうか。

手段：聞き込み・物的証拠・DB照会・民俗学文献・SNS追跡・直接観察。
⚠ 調査なしの討伐は自殺行為
02
解明プロセス
怪異のルールを特定し弱点につながる法則を見つける段階。解明プロセスそのものが命がけだ。

ルールを破れば逃げられなくなる。情報収集と生存のトレードオフが常に発生する。

改造個体か別個体かの判断がここで命運を分ける。
⚠ 解明なしの討伐は自殺行為
03
討伐プロセス
調査と解明で得た情報を元に怪異を無力化する段階。核を破壊または無効化することが基本だが、核の在処と無効化の方法は怪異によって異なる。

物理と霊的手段はどちらも基本的に有効だが怪異によって耐性がある。
⚠ 戦闘力だけで解決できる怪異は存在しない
討伐における耐性と制約
タイプ有効な手段無効な手段
物理耐性型霊的手段・魔法・魔導具物理攻撃がほぼ通じない
霊的耐性型物理・魔導具魔法や霊的手段が通じない
ルール依存型怪異のルールに則った手段のみ解明なしでは全ての攻撃が無効になる
06 — Ability System

能力体系 ABILITIES

Type A — チートコード
魔法（チートコード）
世界のソースコードに直接介入する技術。素養が必要であり訓練によって習得する。再現性が高い。電気やガスと同等のインフラとして位置づけられており資格制度と安全基準が存在する。

大きな魔法を使えば使うほど大きな怪異が生まれる可能性が跳ね上がる。魔法で怪異を倒すことと魔法で怪異を生むことは表裏一体だ。
Type B — アプリケーション
異能（アプリケーション）
怪異と同じ発生原理の延長線上にある力で、強烈な体験・執着・信念が人間の内側で生み出す。直感的に扱えるが制御が難しい。素養の有無に関わらず発現する可能性がある。

使い続けると使用者が怪異に近づいていく（侵食）リスクがある。 その先に何があるかは誰も正確には把握していない。
項目魔法（チートコード）異能（アプリケーション）
原理外部介入内部発生
再現性高い低い
習得訓練と素養体験と執着
リスク怪異を生む使用者の変質（侵食）
社会的扱いインフラ・資格制度あり管理しきれていない
侵食（Erosion）： 異能や特殊な装備（怪異核素材）を長期間使用することで使用者が怪異に近づいていく現象。どこまで進むと取り返しがつかなくなるのかは不明だ。侵食が進んだ討伐者が討伐対象になった事例があるという話が無所属の間に流れている。
魔導具

魔力を体内から吸い出し特定の効果に変換する道具だ。素養のない人間でも使える。あらかじめ設定された効果しか出せないが安定している。製造には素養持ちの技術者が必要で高品質なものほど高価だ。

怪異核素材の魔導具： 闇市場には怪異の核を素材として組み込んだ魔導具が存在するという情報がある。使用者に何らかの代償が生じるとされているが詳細は不明だ。侵食が進む可能性がある。関わることは推奨しない。
覚醒パターン
パターン概要特徴・リスク
先天覚醒型生まれつき素養を持ち訓練で開花祓部に多い。安定した能力基盤
ショック覚醒型怪異に関わる強烈な体験が引き金恨みが動機に直結しやすい。精神的ダメージを抱えやすい
実験覚醒型何者かによる人体実験で強制覚醒素養強度が高い代わりに代償が大きい。追われている可能性がある
接触覚醒型怪異の核や特殊素材への長期接触で覚醒侵食リスクが伴う。変質が進む可能性がある
素養について： 素養の芽自体は人口の30〜40%に潜在しているが、実際に使えるレベルまで開花するかどうかは訓練・環境・きっかけに大きく左右される。素養持ちの子供を巡って国家・企業・正体不明の組織が激しく争奪している。
07 — Equipment System

装備分類 EQUIPMENT

討伐者の装備は六種に分類される。全ての装備に魔導具を組み込むことが可能だが、規模に比例して怪異発生リスクが生まれる。

武装型WEAPON TYPE
武器や盾など手に持って使用する武装の総称。最もシンプルで最も歴史が長い分類。剣・銃・槍のような物理武器から魔導具を組み込んだ魔法武器まで幅が広い。携行性が高く状況に応じた持ち替えが容易だ。素養の有無に関わらず使用できる。
