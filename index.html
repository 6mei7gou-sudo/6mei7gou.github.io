01 — World Overview

世界概要 WORLD OVERVIEW

舞台は近未来の架空日本だ。地名も地図も現実とは異なるが、文化・言語・民俗学的背景は日本のそれを色濃く引き継いでいる。表層はサイバーパンク的な高度情報社会であり、魔法はガスや電気と同等のインフラとして社会に組み込まれている。

Surface
高度情報社会
魔法が資格制度と安全基準を持つインフラとして機能する近未来。SNSと都市伝説が怪異の燃料になる時代だ。
Underside
怪異の侵食
噂や言説が質量を持ち、現実にバグとして侵食する怪異が日常的に発生している。一般市民には秘匿されている。
Paradox
見えるが信じられない
怪異は一般人にも見える。干渉もできる。問題は「話しても誰も信じてくれない」ことだ。
秘匿と認知： 怪異の存在は秘匿されているが、見えないということとは別だ。知識・訓練・装備の差が討伐者と一般人を分ける。無知が人を殺す。
権力の三極構造
極構成行動原理
第一極国家・蒼鉄機工（国家系企業）秩序の維持。怪異の存在を秘匿することで秩序が成立する
第二極独立系企業（雷禽重工等）・傭兵集団利益。国家にも危険な勢力にも必要とされる立場を維持して生き残る
第三極正体不明の組織既存の権力構造の外側。怪異そのものを利用する勢力が存在するという情報がある
無所属の立場： 三極のどこにも属さない。三極すべてから必要とされ、三極すべてから敵視される。
情報統制

公的機関は怪異による被害を魔法設備の事故・ガス漏れ・集団ヒステリーとして処理する。SNSの目撃情報は陰謀論扱いで埋もれるよう誘導される。まれに記憶に直接介入する手段が使われるという話もあるが詳細は不明だ。

「あなたの体験は本物だ」
— 謎の組織が怪異の被害者に語りかける言葉。最大限の注意が必要だ。

03 — Kaijin System

怪異について KAIJIN

怪異とは、集合的な噂・言説・信念が臨界点を超えた時に現実へと侵食する存在だ。専門家はこれをバグと呼ぶ。全ての怪異は核とルールを持つ。

Core — 核
核とは何か
怪異の存在の中心であり消滅条件の鍵だ。基本的には物に宿る。特定の場所・道具・建造物・記録媒体などが核になる。稀に死体や生きた人間に宿る場合があり、対処の難度と倫理的複雑さが跳ね上がる。

核を無効化・破壊することが怪異消滅の最短経路だが、どこに何が核なのかは調査と解明を経て初めて分かる。
Rule — ルール
ルールとは何か
怪異が人間を害するための条件だ。ルールを破れば破るほど怪異から逃げられなくなる。

ルールは怪異が人を殺す手順であると同時に、怪異が人を捕捉・追跡するトリガーでもある。無意識にルールを一度も破らなかった一般人が理由も分からず生還するケースがある。

強い怪異ほどルールが複雑で厳密になる傾向がある。
怪異と神について： 討伐者の間ではこういう話がある。「恐れられているか信仰されているかの違いだけだ。」真偽は不明だが、神社の御神体が怪異化したという記録が残っていることは事実だ。神だから討伐できないという保護はこの世界に存在しない。
改造個体と別個体
Modified — 改造個体
ルール変質 50%未満
元の核とルールが主体として残っている。既存の知識がある程度通用するが、ある一点でルールが決定的に食い違う瞬間が来る。調査段階では「文献との齟齬・被害パターンの不一致・核の所在の混乱・目撃証言の分裂」という四つの違和感として現れる。
New Entity — 別個体
ルール変質 50%以上
元の知識はほぼ役に立たない。 完全に別の怪異として扱う。最も恐ろしいパターンは「改造個体と解明して討伐に挑んだ瞬間に別個体だと判明するケース」だ。怪異は健在で討伐者はルールを破り続けた状態で無防備になっている。
04 — Classification

等級体系 GRADE SYSTEM

怪異は存在強度等級と脅威度種別の二軸で表記される。二つの軸は独立しており、必ずしも一致しない。

存在強度等級（特級〜五級）
等級基準特徴
特級現状の手段では消滅が極めて困難封印・管理が限界。鵺ヶ原事変の怪異が該当
一級信念密度が非常に高い強力な怪異長い歴史を持つ。対処法が民俗学記録に存在する場合が多い
二級記録のある古い怪異の大半訓練された討伐者であれば対処可能
三級拡散が進んだ新しい怪異などSNS時代に急成長するケースがある
四級発生したばかりの新しい怪異初見殺しが最大のリスク
五級魔法事故由来の微小怪異自然消滅することが多い
脅威度種別（甲種〜丁種）
種別基準対処の考え方
甲種無差別に人を害する。接触した全員が標的即時対応が必要
乙種ルールを破った者など条件付きで害するルール遵守で生存できる
丙種刺激しなければ安全放置が最善の場合がある
丁種人を害さない。共存可能または信仰対象に近い討伐より封印・共存の選択肢もある
重要な注意： 強いが安全な怪異（一級丁種）、弱いが危険な怪異（五級甲種）が存在する。等級だけで危険度を判断するな。さらに影響範囲（広域型・局所型）と被害性質（物理型・精神型・複合型）が付記される。
永続性による分類
Type A
一時型
核が破壊されると消滅する標準的な怪異。ただし噂の中身が変質して似た性質の派生体が後日発生するケースがある。派生体は元の怪異と別個体として扱え。
Type B
循環型（都市伝説型）
社会に深く根を張った怪異。核を破壊しても噂が生き続ける限り新たな核を形成して再出現する。討伐は根絶ではなく制圧だ。
Type C — 破壊困難
定着型（土地型）①
核が土地と一体化しすぎており通常手段では無効化できない。大規模な魔法での破壊はさらに大きな怪異を生むジレンマがある。公的機関が封鎖して放置する理由がここにある。
Type C — 一度限り
定着型（土地型）②
核の特定と破壊に成功すると噂の構造ごと崩壊して完全消滅する。達成した討伐者は伝説扱いされる。白銀の廃線事件はこのタイプへの対処事例として語り継がれている。
05 — Exorcism Process

討伐プロセス PROCESS

討伐は三段階のプロセスで構成される。戦闘はあくまで最終段階であり、その前の二段階が生死を分ける。

01
調査プロセス
怪異の存在を確認し正体と性質を特定する段階。

明らかにすること：この怪異は何者か、古い怪異か新しい怪異か、核はどこに宿っているか、被害のパターンと範囲はどうか。

手段：聞き込み・物的証拠・DB照会・民俗学文献・SNS追跡・直接観察。
⚠ 調査なしの討伐は自殺行為
02
解明プロセス
怪異のルールを特定し弱点につながる法則を見つける段階。解明プロセスそのものが命がけだ。

ルールを破れば逃げられなくなる。情報収集と生存のトレードオフが常に発生する。

改造個体か別個体かの判断がここで命運を分ける。
⚠ 解明なしの討伐は自殺行為
03
討伐プロセス
調査と解明で得た情報を元に怪異を無力化する段階。核を破壊または無効化することが基本だが、核の在処と無効化の方法は怪異によって異なる。

物理と霊的手段はどちらも基本的に有効だが怪異によって耐性がある。
⚠ 戦闘力だけで解決できる怪異は存在しない
討伐における耐性と制約
タイプ有効な手段無効な手段
物理耐性型霊的手段・魔法・魔導具物理攻撃がほぼ通じない
霊的耐性型物理・魔導具魔法や霊的手段が通じない
ルール依存型怪異のルールに則った手段のみ解明なしでは全ての攻撃が無効になる
06 — Ability System

能力体系 ABILITIES

Type A — チートコード
魔法（チートコード）
世界のソースコードに直接介入する技術。素養が必要であり訓練によって習得する。再現性が高い。電気やガスと同等のインフラとして位置づけられており資格制度と安全基準が存在する。

大きな魔法を使えば使うほど大きな怪異が生まれる可能性が跳ね上がる。魔法で怪異を倒すことと魔法で怪異を生むことは表裏一体だ。
Type B — アプリケーション
異能（アプリケーション）
怪異と同じ発生原理の延長線上にある力で、強烈な体験・執着・信念が人間の内側で生み出す。直感的に扱えるが制御が難しい。素養の有無に関わらず発現する可能性がある。

使い続けると使用者が怪異に近づいていく（侵食）リスクがある。 その先に何があるかは誰も正確には把握していない。
項目魔法（チートコード）異能（アプリケーション）
原理外部介入内部発生
再現性高い低い
習得訓練と素養体験と執着
リスク怪異を生む使用者の変質（侵食）

侵食（Erosion）： 異能や特殊な装備（怪異核素材）を長期間使用することで使用者が怪異に近づいていく現象。どこまで進むと取り返しがつかなくなるのかは不明だ。侵食が進んだ討伐者が討伐対象になった事例があるという話が無所属の間に流れている。
魔導具

魔力を体内から吸い出し特定の効果に変換する道具だ。素養のない人間でも使える。あらかじめ設定された効果しか出せないが安定している。製造には素養持ちの技術者が必要で高品質なものほど高価だ。

怪異核素材の魔導具： 闇市場には怪異の核を素材として組み込んだ魔導具が存在するという情報がある。使用者に何らかの代償が生じるとされているが詳細は不明だ。侵食が進む可能性がある。関わることは推奨しない。
覚醒パターン
パターン概要特徴・リスク
先天覚醒型生まれつき素養を持ち訓練で開花祓部に多い。安定した能力基盤
ショック覚醒型怪異に関わる強烈な体験が引き金恨みが動機に直結しやすい。精神的ダメージを抱えやすい
実験覚醒型何者かによる人体実験で強制覚醒素養強度が高い代わりに代償が大きい。追われている可能性がある
接触覚醒型怪異の核や特殊素材への長期接触で覚醒侵食リスクが伴う。変質が進む可能性がある
素養について： 素養の芽自体は人口の30〜40%に潜在しているが、実際に使えるレベルまで開花するかどうかは訓練・環境・きっかけに大きく左右される。素養持ちの子供を巡って国家・企業・正体不明の組織が激しく争奪している。

08 — Three Factions

三種の討伐者 FACTIONS

⛩
公的機関（祓部）
HARAEBU — PUBLIC AGENCY
魔導省の内部組織として存在する国家機関の怪異対処部門。警察や自衛隊の入隊検査で素養を持つ人間が選別されて配属される。古い怪異に関する深い知識と専門的な訓練体系を持つ。

祓部（はらえべ）詳細 PUBLIC AGENCY

公的な怪異対処組織。魔導省の内部組織として存在し、外部からは「魔導省の特殊部署」として認識されている。終戦後怪異爆発期（1947年）の対処が追いつかない状況の中で組織化された。

第一部門：対処部

古怪班： 古い怪異専門のチーム。民俗学的知識と古来の手法に精通。御幣・塩・酒・榊のような古来の道具を実際の怪異対処に使用する。これらは単なる儀式用具ではなく古い怪異のルールに干渉するための有効な手段だ。

新怪班： 新しい怪異専門のチーム。SNS・ネットロア・都市伝説の調査に特化。若い隊員が多く祓部の中で最も現代的な部署。古怪班との文化的摩擦が生じやすい。

封印班： 禁足地の管理と封印処理に特化したチーム。白銀の廃線事件のような大型封印案件では対処部全体が動く。

INTELLIGENCE DIVISION

第三部門：技術部
TECHNICAL DIVISION

装備の整備・改修・開発を担う部門。蒼鉄機工との共同開発窓口でもある。技術士・術式士・機動兵装士の三種の専門職種を持つ。

技術部は祓部の中で最も「普通の職場」に近い雰囲気を持つ。神職的な文化が薄く実用主義が優先される。現場の無理な要求と安全基準の間で板挟みになることが多い。

10 — Mercenary Details

傭兵詳細 MERCENARIES

怪異の討伐を仕事にしている討伐者のうち、国家にも特定の組織にも属さず報酬を対価に案件を受ける者の総称だ。傭兵という呼称は業界内での慣用的な呼び方であり法的な定義はない。同じ傭兵でも大企業がバックについた精鋭部隊もあれば個人で受注して動く一匹狼もある。規模も専門性も報酬体系も千差万別だ。

祓部との違い
義務か、選択か
祓部の討伐者は組織の命令と方針に従う義務がある。傭兵は案件を選べる。断れる。撤退できる。

その自由の対価として法的保護がなく安定した補給もなく組織の後ろ盾もない。 現場で何があっても自己責任だ。
業界での生存
実績と信頼が資本
傭兵が業界で生き残るための資本は二つだ。実績と信頼だ。どんな案件をこなしてきたか、約束を守るかどうか、現場で何をする人間か。それが傭兵の価値を決める。

二つ名はその実績が形になったものだ。 二つ名を持つ傭兵は一人の人間であると同時に業界における一つのブランドでもある。
強み： 法的制約を受けない行動の自由 / バック企業によっては高性能・試作品装備へのアクセス / 複数の組織と関係を持てる立場 / 案件を選べる自由

制約： 法的保護が薄く犯罪者扱いされるリスクがある / バック企業の方針に縛られる場合がある / 収益が安定しない / 祓部との摩擦が発生しやすい
依頼人と案件の仕組み
依頼人の種類
誰が傭兵を雇うか
依頼人は多様だ。個人・企業・地方自治体はもちろん、場合によっては祓部自身が人員不足を補うために傭兵を雇うこともある。「公的機関が民間に頼む」という構図は表向きには存在しないが、現場レベルでは珍しくない。

企業が依頼人の場合、解決よりも「情報の隠蔽」や「事態の収束」を優先する条件を付けることがある。依頼内容の全貌は最初から開示されないことも多い。
案件の受け方
どう仕事を得るか
直接依頼・紹介・仲介業者経由の三パターンがある。大規模集団は独自の受注窓口を持つが、個人傭兵は信頼できる仲介者との関係が命綱になる。

案件の情報は不完全なことが多い。 現場に入ってから想定と違う怪異に遭遇することは珍しくなく、そのための撤退判断と再交渉の能力も傭兵の実力のうちだ。

傭兵と祓部の関係は複雑だ。対立・協力・無視・監視が案件や人間関係によって混在する。傭兵を犯罪者に近い存在として扱う祓士がいる一方、現場では傭兵の実力と知識を頼りにする祓士も多い。公式な関係は存在しないが非公式な協力は日常的に行われている。

企業詳細 COMPANIES

魔導具産業は国家系企業と独立系企業の二極構造を持つ。国家系は安全・保守・安定、独立系は高性能・高リスク・尖りという非対称性が産業全体の基盤をなす。

蒼鉄機工（そうてつきこう）
国家系企業 / 設立：1900年代初頭 / 主取引先：祓部・国家機関

この世界最古の魔導具製造企業。「安全基準を作る側」の立場を維持してきた。製品の性能より安定性と安全基準を優先する社風が徹底している。社内では「怪異を生まない製品を作る」という言葉が合言葉として使われている。

主要製品ライン
  • 標準魔導武装シリーズ祓部への標準支給品。剣・銃・補助装置の三種が基本構成で素養の有無に関わらず安定した出力を得られる設計。
  • 鉄甲シリーズ（半装身型・全装身型）祓部向けに設計された装身型装備。安全基準を最優先で設計。全装身型の鉄甲壱式は2019年の事故以降改良版の弐式が開発中。
  • 蒼鉄機動シリーズ（戦闘用搭乗型）安全優先設計の機動兵器。最大出力は独立系に劣るが長時間稼働時の安定性と整備性が強み。
内部の問題

開発部門（性能向上を求める）と安全管理部門（基準の維持を求める）の対立が慢性化。2019年の事故以降、安全管理部門の発言力がさらに強化された。獅子倉・鉄也のような告発者が生まれる背景がここにある。

独立系企業（最大手）/ 設立：1973年 / 主取引先：傭兵集団

独立系企業の中で最大の規模と技術力を持つ。「性能が全て」という哲学が社風の核心にある。安全基準は守る。しかし基準の範囲内で可能な限り尖った製品を作る。傭兵の現場からのフィードバックを製品開発に直接反映するサイクルが確立されている。

第二章：怪異
怪異★ 怪異▼
集合的な噂・言説・信念が臨界点を超えた時に現実へと侵食する存在の総称。全ての怪異は核とルールを持つ。核は怪異の存在の中心であり消滅条件の鍵。ルールは怪異が人間を害するための条件だ。怪異は討伐者と一般人を区別しない。知識・訓練・装備の差が生存率を変える。
関連：バグ、核、ルール、信念密度
核★ 怪異▼
怪異の存在の中心であり消滅条件の鍵となる部位。基本的には物に宿る。特定の場所・道具・建造物・記録媒体などが核になる。稀に死体や生きた人間に宿る場合があり対処の難度と倫理的複雑さが跳ね上がる。核を無効化・破壊することが怪異消滅の最短経路だが、どこに何が核なのかは調査・解明を経て初めて分かる。
関連：怪異、破壊、怪異核素材
ルール★ 怪異▼
怪異が人間を害するための条件。怪異はこのルールに従ってしか人間を害せない。ルールを破れば破るほど怪異から逃げられなくなる。ルールは怪異が人を殺す手順であると同時に怪異が人を捕捉・追跡するトリガーでもある。無意識にルールを一度も破らなかった一般人が理由も分からず生還するケースがある。
関連：怪異、解明プロセス
改造個体★ 怪異分類▼
元の怪異のルールが変質しているが変質割合が50%未満の怪異。既存の知識がある程度通用するが決定的にルールが食い違う一点で驚きが生じる。調査段階では文献との齟齬・被害パターンの不一致・核の所在の混乱・目撃証言の分裂という四つの違和感として現れる。
